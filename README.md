# MENEN SZYMON — Site CV

Site personnel / carte de visite en ligne, en constante évolution depuis 2021.

## Stack

HTML5 / CSS3 / JavaScript vanilla — aucun framework, aucun build, aucune dépendance.
Hébergé sur GitHub Pages.

## Historique

Ce projet a évolué au fil des années, passant de simple page CV statique à un site interactif avec :

- Dark / light mode automatique
- Scroll snap sur mobile
- Layout wireframe responsive
- Animations CSS (typewriter, scroll-reveal, marquee)
- Portfolio en construction

Chaque version reflète ma progression en tant que développeuse.
